# 00018 - Double Feature Night

## Problem
Sometimes couples want to watch two movies in one night but don't know what pairs well together.

## Solution
Suggest complementary double features:
- "Double Feature" mode in room settings
- Algorithm suggests pairs that work well together:
  - Same director/actor
  - Thematic connections (heist movies, time travel, etc.)
  - Complementary tones (action + comedy)
  - Classic + modern remake
- Both users swipe on the *pair*, not individual movies
- Match requires both to agree on the combination

## Value
- Solves a real planning problem for movie marathons
- Shows off movie knowledge/curation
- Creates longer engagement sessions
- Differentiates from other movie apps

## Success Criteria
- [ ] Pairing algorithm with 3-5 pairing strategies
- [ ] UI shows both movies side-by-side
- [ ] Swipe applies to the pair
- [ ] Match adds both to queue in suggested order
- [ ] Explain *why* these pair well ("Same director: Nolan")

## Assumptions to Validate
- Couples actually do double features
- They want suggestions vs picking themselves
- Pairs feel natural and complementary
- Doesn't complicate the simple swipe UX too much
