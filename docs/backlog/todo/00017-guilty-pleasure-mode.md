# 00017 - Guilty Pleasure Mode

## Problem
Users might hesitate to swipe right on movies they secretly love but are embarrassed about (cheesy rom-coms, bad action movies, etc.).

## Solution
"Guilty Pleasure Mode" - anonymous swiping:
- Toggle mode in settings
- Swipes are hidden from partner until there's a match
- Only matched movies are revealed
- Fun icon/badge: "🙈 Guilty Pleasure Match!"
- Optional: reveal guilty pleasures after X matches

## Value
- Increases honest swiping behavior
- Creates fun surprise moments when guilty pleasures match
- Removes social pressure from movie selection
- Leads to more authentic matches

## Success Criteria
- [ ] Add toggle in user settings
- [ ] Hide swipe activity when mode is active
- [ ] Only reveal on mutual match
- [ ] Special badge/animation for guilty pleasure matches
- [ ] Track how often this mode is used (analytics)

## Assumptions to Validate
- Users actually self-censor their swipes
- Anonymity increases honest engagement
- The "reveal" moment is delightful vs awkward
- Both partners are comfortable with this feature
