# 00016 - Movie Night Streaks

## Problem
No way to track consistent movie-watching habits or celebrate the couple's engagement with the app.

## Solution
Gamify movie nights with streak tracking:
- Track consecutive weeks with at least one match
- Display streak count on dashboard
- Celebrate milestones (5, 10, 25, 50, 100 nights)
- Show "longest streak" achievement
- Optional push notifications: "Don't break the streak!"

## Value
- Encourages regular use of the app
- Creates habit formation
- Adds fun competition/challenge element
- Strengthens couple bonding through shared ritual

## Success Criteria
- [ ] Track when matches occur (timestamp)
- [ ] Calculate current streak (weekly basis)
- [ ] Display streak prominently in UI
- [ ] Show milestone celebrations with animations
- [ ] Store historical "longest streak" data

## Assumptions to Validate
- Streaks motivate continued use vs feeling like pressure
- Weekly cadence is right (vs daily/monthly)
- Couples care about this metric
