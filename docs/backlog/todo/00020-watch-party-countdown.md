# 00020 - Watch Party Countdown

## Problem
After matching on a movie, there's no help with the "when do we watch it?" coordination.

## Solution
Countdown timer for scheduled movie nights:
- After match: "Schedule watch time?"
- Pick date/time together
- Countdown displayed on dashboard: "Movie night in 2 days, 3 hours!"
- Push notification 1 hour before: "🍿 Movie night starting soon!"
- Optional: "Start watching" button that opens streaming link
- Post-watch: Prompt to rate the movie together

## Value
- Converts matches into actual viewing events
- Reduces procrastination ("we matched but never watched it")
- Adds anticipation/excitement
- Closes the loop from match → watch → feedback

## Success Criteria
- [ ] Schedule UI after match
- [ ] Store scheduled time in database
- [ ] Display countdown on dashboard
- [ ] Send notifications at T-1 hour
- [ ] "Start watching" CTA at scheduled time
- [ ] Post-watch rating prompt

## Assumptions to Validate
- Couples want this level of structure
- Scheduling doesn't feel too rigid
- Notifications are welcome vs annoying
- Post-watch rating has enough value to users
