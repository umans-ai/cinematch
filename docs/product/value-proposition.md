# Value Proposition

## Problem

Couples waste 30+ minutes scrolling through streaming services, disagreeing on what to watch, ending up watching nothing or settling for something neither really wants.

## Solution

CineMatch turns movie selection into a collaborative game:
- Create a room in 2 seconds (no accounts)
- Both swipe independently on the same curated list
- Get instant notification when you both like the same movie
- Decision made in under 5 minutes

## Key Differentiators

| Aspect | Traditional | CineMatch |
|--------|-------------|-----------|
| Setup | Create accounts, share passwords | 4-digit code, no accounts |
| Process | Endless discussion, compromise | Independent swiping, data-driven match |
| Time to decision | 30+ minutes | < 5 minutes |
| Discovery | Endless catalogs, paradox of choice | Curated 50-movie list, good enough |

## Target Users

- Couples who stream together regularly
- Friends choosing a movie for movie night
- Long-distance couples watching remotely (future: sync playback)

## Success Metrics

- Time from room creation to match: target < 5 minutes
- Completion rate: % of rooms that find a match
- Return rate: % of users who create multiple rooms
